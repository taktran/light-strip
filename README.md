# Light strip

Light playground

## Usage

Install dependencies

    npm install

Set up lights

1. Upload [custom firmata firmware using arduino software](https://github.com/ajfisher/node-pixel/tree/master/firmware#using-arduino-ide)
2. Run lights

  ```sh
  npm run lights
  ```
